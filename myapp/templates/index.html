1. Two Sum
Medium

Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target.

You may assume that each input would have exactly one solution, and you may not use the same element twice.

Example 1:
Input: nums = [2,7,11,15], target = 9 Output: [0,1] Explanation: Because nums[0] + nums[1] == 9, we return [0, 1].
Example 2:
Input: nums = [3,2,4], target = 6 Output: [1,2]

Which method is used to add one or more elements to the end of an array in JavaScript?
A) array.append() - Adds elements to the end
B) array.push() - Adds elements to the end and returns new length
C) array.insert() - Inserts elements at specified position
D) array.concat() - Merges arrays without modifying original
